This example tests the accessible name calculation used for the role of region. This example includes a div element with the role attribute set to region. It is then labelled using the aria-labelledby attribute, the aria-label attribute, and the title attribute. The aria-labelledby attribute is set to the id of the h1 element (which is "Mourning Dove".) The value of the aria-label attribute is set to "Bird Species". The value of the title attribute is set to "Bird Species-Mourning Dove".

According to the specification, a screen reader should assign "Mourning Dove" as the label for the region.
